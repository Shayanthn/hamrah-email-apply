### ⚡ نصب سریع

```bash
# ۱. کلون کردن

# ۲. ورود به پوشه
cd hamrah-email-apply

# ۳. نصب وابستگی‌ها
npm install

# ۴. اجرای برنامه
npm start

# 🎉 برنامه در آدرس http://localhost:3000 اجرا می‌شود!
```
